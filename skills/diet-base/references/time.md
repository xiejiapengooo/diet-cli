# +time

用于把用户口语时间转换为 CLI 所需绝对时间格式：`YYYY-MM-DD HH:mm`。

## 规则

1. 先以用户当前时区解释时间词（若未给出时区，默认使用会话时区）
2. 再换算成绝对日期时间字符串，传给 `--at`、`--from`、`--to`
3. 执行后在回复中回显本次使用的绝对时间，避免歧义

## 映射示例（当前日期：2026-04-02）

- "今天早餐 8 点" -> `2026-04-02 08:00`
- "明天中午 12:30" -> `2026-04-03 12:30`
- "昨天晚上 7 点" -> `2026-04-01 19:00`

## 查询范围示例

- "查今天吃过的"：
  - `--from "2026-04-02 00:00"`
  - `--to "2026-04-02 23:59"`
- "查昨天晚餐相关记录"：
  - `--meal dinner`
  - `--from "2026-04-01 00:00"`
  - `--to "2026-04-01 23:59"`
